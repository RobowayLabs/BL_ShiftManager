# Banglalink Shift Manager

Enterprise-grade NOC dashboard for monitoring shifts, AI-driven alerts, and employee performance.

## Quick Start
1.  **Dashboard**: View real-time system health and AI alert trends.
2.  **Shift Planner**: Assign employees to Morning, Day, or Night shifts on a calendar.
3.  **Shift Viewer**: Check the next 7 days of scheduled shifts side-by-side.
4.  **Employees**: Manage staff and generate individual activity reports.
5.  **Reports**: Review AI-captured metadata (Start/End/Break times) and behavioral alerts (Drowsiness, Sleep, Mobile Usage, Absence).
